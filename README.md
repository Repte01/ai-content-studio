# AI Content Studio

Aplicación de procesamiento de imágenes con IA que incluye:
- OCR y traducción de texto
- Generador de contenido para redes sociales
- Asistente IA para descripciones
- Galería de imágenes
- Panel de usuario con estadísticas

## Despliegue en Streamlit Cloud

1. Sube este repositorio a GitHub
2. Conéctalo a Streamlit Cloud
3. Configura la API Key de Gemini en Secrets
4. ¡Listo! Tu app estará disponible públicamente
